# Salamoonder Examples

Welcome to the Salamoonder Examples repository! This collection showcases practical examples and use cases for the Salamoonder SDK.

## About

This repository contains a curated set of examples demonstrating how to build with the [Salamoonder Python SDK](https://github.com/Salamoonder-LLC/salamoonder-py). Whether you're just getting started or looking for inspiration on how to solve specific problems, you'll find working code samples here.

## Getting Started

### Prerequisites

- Python 3.7+
- The Salamoonder SDK

### Installation

Clone both the Salamoonder SDK and this examples repository:

```bash
git clone https://github.com/Salamoonder-LLC/salamoonder-py.git
git clone https://github.com/Salamoonder-LLC/salamoonder-examples.git
```

Make sure the `salamoonder` folder from `salamoonder-py` is available locally, and you're all set to run the examples!

## Examples

This repository contains examples for solving various bot detection and security challenges:

- **Akamai**: Web, SBSD
- **Datadome**: Interstitial, Slider
- **Incapsula**: Reese84, UTMVC
- **Kasada**

Explore the various examples in this repository to learn how to use Salamoonder for your use cases. Each example includes clear documentation and is ready to run.

## Contributing

We'd love to see your examples! If you've built something interesting with Salamoonder, we encourage you to share it with the community.

### How to Contribute

1. Fork this repository
2. Create a new branch for your example (`git checkout -b add-my-example`)
3. Add your example code with clear documentation
4. Submit a pull request

We'll review your submission and merge it into the repository. Thanks for helping grow the Salamoonder community!
